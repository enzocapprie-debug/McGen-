# McGen+ — SaaS Mockup Generator

A professional SaaS application for generating stunning 3D device mockups and eCovers entirely in the browser.

## Features
- **9 Categories**: Books, Boxes, Phones, Tablets, Laptops, Desktops, Browsers, Websites, Multi-Device.
- **27+ Templates**: High-quality CSS/SVG-based 3D templates with zero asset weight.
- **Real-time Canvas Editor**: Scale, reposition, and toggle backgrounds instantly.
- **100% Client-Side Processing**: No servers needed for image processing (free to scale).
- **No Login Wall**: Fully open and public access for all users.

## Setup Instructions (Vercel/Local)

1. Clone or download this repository.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the local development server.
4. Or push to GitHub and import the project in [Vercel](https://vercel.com) for a 1-click deployment.

## Tech Stack
- Next.js 14 (App Router)
- Tailwind CSS
- Framer Motion
- html2canvas (Export engine)
